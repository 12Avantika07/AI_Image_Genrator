# 🎨 AI Image Generator

An AI-powered Image Generator built using **Python**, **Streamlit**, and **Pollinations AI API**. This application allows users to generate stunning AI-generated images simply by entering a text prompt and selecting their preferred art style.

---

## 🚀 Features

- 🎨 Multiple Art Styles
  - Photorealistic
  - Anime
  - Vintage Victorian
  - Sketch
  - Fantasy
  - Digital Art
  - Oil Painting
  - Watercolor
- 📝 Generate images from text prompts
- 📏 Adjustable image width and height
- ⚡ Fast image generation using Pollinations AI
- 🌙 Modern dark-themed UI
- ✅ Displays API status code
- 🖼️ Instant image preview

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Requests
- Pollinations AI API

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/AI-Image-Generator.git
```

Move into the project directory

```bash
cd AI-Image-Generator
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run image.py
```

## 🎯 How to Use

1. Open the application.
2. Select your preferred Art Style.
3. Adjust Image Width and Height.
4. Enter a text prompt.
5. Click **Generate Image**.
6. Wait a few seconds.
7. The AI-generated image will appear on the screen.
